# Places to Visit Around Nanaimo

A travel-themed multi-page website that highlights scenic destinations around Nanaimo, British Columbia. The site is designed to promote local tourism through images, descriptions, a gallery, and a simple planning page.

## Project Overview

This website showcases beautiful places to visit in and around Nanaimo, including parks, beaches, lakes, islands, and hiking spots. It was built as a front-end web project using HTML and CSS, with a small JavaScript effect on the homepage hero section.

## Main Features

- **Video hero header** on the homepage
- **Scroll fade effect** using JavaScript
- **Multi-page navigation**
- **Destination highlights** with images and descriptions
- **Plan Your Day** table for quick travel ideas
- **Gallery page** with a grid of travel photos
- **About Us page** introducing the purpose of the site
- **Language selector** and **feedback form**
- External links for:
  - Nanaimo events
  - BC Ferries booking

## Pages Included

- `miniproject.html` or homepage  
  Main landing page with a background video, Nanaimo introduction, and call-to-action.

- `places.html`  
  A destination page featuring places such as:
  - Sugar Loaf Mountain
  - Colliery Dam Park
  - Neck Point Park
  - Little Mountain
  - Little Qualicum Falls
  - Cameron Lake
  - Gabriola Island
  - Cox Bay Beach

- `plan.html`  
  A table-based trip planning page with destination types, reasons to visit, and best activities.

- `gallery.html`  
  A photo gallery showing images from the featured travel spots.

- `aboutus.html`
  A short overview of the website's purpose and why Nanaimo is worth visiting.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Design Elements

- Responsive layout basics using the viewport meta tag
- Embedded background video
- Image galleries with grid and flex layouts
- Styled navigation bar and links
- Travel-focused color themes and typography

## Project Goal

The goal of this website is to encourage people to explore Nanaimo and nearby destinations by presenting useful travel ideas in a visually engaging way.

## How to Run the Project

1. Download or copy all project files into one folder.
2. Make sure the folder includes:
   - all HTML files
   - the 'css' folder
   - the 'img' folder
   - the video file used on the homepage
3. Open the homepage file in a web browser.
